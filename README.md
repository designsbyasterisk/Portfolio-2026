# Shreya Kulkarni - Portfolio Website

Welcome to the repository for my professional portfolio website. This site showcases my UI/UX designs, product design case studies, and engineering projects. It is designed to be fully responsive, lightweight, and engaging.

## Tech Stack

*   **Core:** HTML5, CSS3, JavaScript (ES6+)
*   **Animations:** GSAP (GreenSock Animation Platform) & ScrollTrigger
*   **Aesthetics:** Modern, premium styling with custom cursors, smooth transitions, and fluid grid layouts.

## Project Structure

*   `index.html` - The landing page and homepage.
*   `resume.html` - My professional resume layout.
*   `project-details/` - In-depth case study layouts for specific projects (e.g. F1 Dashboard, Honee, Restease).
*   `Projects/` - Individual project demo links and standalone builds.
*   `css/` & `js/` - Source stylesheets and animation scripts.

## Local Development

To run this project locally, simply clone the repository and open `index.html` in your browser. Alternatively, you can use a local development server like **Live Server** in VS Code:

1. Clone the repository:
   ```bash
   git clone https://github.com/designsbyasterisk/Portfolio-2026.git
   ```
2. Open the directory in your code editor.
3. Launch your local server.
